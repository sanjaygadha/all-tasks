EMMETS

Emmets are shortcut abbrevations for writing the HTML and CSS code.

EMMET NAME DISCRIPTION
Child: > ">" operator to nest elements inside each other
Sibling: + "+" operator to place elements near each other,on the same level
Climb-up: ^ "^" operator descending down the genetared tree and the positions of all sibling elements
Grouping: () "()" used to grouping subtrees in complex abbrevations
Multiplication: * "*" used to specify how many times an element should be written
Addition: + "+" used to write a complete tag in short
Item numbering: $ "$" used to print the current number of repeated elements
HTMl: ! "!" used to create the default body for HTML code
HTML: input It is used to write HTML input elements and define input control
HTML: button It is used to create a button tag in the HTML code,used to define a clickable button
HTML: img It is used to create an image tag in the HTML code,used to define an image
HTML: pic It is used to create a picture tag in the HTML code,used to define containers for multiple image resources
