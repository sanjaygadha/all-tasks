TOPICS DESCRIPTION
HTML HTML stands for Hypertext Markup Language.
It is a standard markup language. It is used to create webpages.
It acts as a structure to the website. It consists of series of elements.
Markup It is set of rules which is used to structure the content, image and layout.
Hypertext The text which reference to another text.
HTML history It was introduced by Tim Berners Lee.
HTML 1-1991 HTML 5-2014
Tags It is a piece of markup language which indicates beginning and end of html element.
Types of tags 1. Container or pair tags: tags contains both starting and ending tag.
2. Empty or void or selfclosing tags: tags doesn't require closing tag.
Anatomy of element It consists of opening tag, some content and closing tag.
Basic structure of HTML HTML should start with DDT(Document DeClaration Type).It tells the document type and version of html and declare at the top of the document.
If we remove delaration type then it will goes to quark mode.
html: is the root element of html document.
head: contais title of the document and contains meta information and external links.
body: contains all the content related to the website.
Heading tag It is used to represent heading for the content.
Heading tags are defined with h1, h2, h3, h4, h5, h6.
Preformatted tag It preserves spaces and format.
Paragraph tag It is used to specify paragraph.
Anchor tag It is used to specify hyperlinks for the document.It is used for internal and external routing.
It takes few attributes href, target, title.
hr tag It is used to insert horizontal line witdin the text context.
br tag It is used to insert line break witdin the text context.
Tables It is used to show the data in tabular format.Tables have rows and cells.
Tables are defined with table tag. Rows are defined witd tr tag. Cells are defined with td tag. Headings are defined with td tag.
Attributes:
Caption, cellpadding, cellspacing, grouping, bgcolor, spanning, widtd and height.

2. Relative path: Path reference to the current document.
